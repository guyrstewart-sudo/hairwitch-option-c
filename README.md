# The Hair Witch — Website Option C (Replica)

**Preview site for Theresa Coggiola / The Hair Witch (Asheville, NC).**
Date: 2026-07-04

This is one of **three** design options built for review. Theresa can view each
one separately and choose the one she likes best — or a blend of the three.

- Option A — Monochrome: stark black/charcoal + gold, all-caps, gallery-forward.
- Option B — Editorial: warmer, magazine-style layout.
- **Option C — Replica** (this repo): a faithful, hand-rebuilt mirror of her
  current live site — her rendered layout, her full 290-photo / 10-video
  archive, category tabs, carousels, and lightboxes reimplemented by hand.

## View it
Once GitHub Pages is enabled for this repo (Settings → Pages → Deploy from
branch → `main` / root), it will be live at:

> https://guyrstewart-sudo.github.io/hairwitch-option-c/
